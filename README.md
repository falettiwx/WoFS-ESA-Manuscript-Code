# WoFS-ESA-Manuscript-Code

Python code to reproduce figures in Faletti et al. (2026) study on ensemble sensitivity analysis (ESA) in the NOAA Warn-on-Forecast System (WoFS).

Accompanying WoFS forecast data are currently stored at the Texas Tech University High-Performance Computing Center (TTU HPCC) and are available upon request. 

For data requests, please contact Billy Faletti at william.faletti@noaa.gov. 

## References:
Faletti, W. L., C. C. Weiss, B. C. Ancell, and P. S. Skinner, 2026: Using Ensemble Sensitivity to Diagnose Environmental Modulation of Mesocyclone Intensity in the Warn-on-Forecast System. _Mon. Wea. Rev._, in review.
